# Content Scout — Заявка в медиа-команду

Опросник для отбора кандидатов на позицию **Контент-скаут / Продюсер соцсетей (Short-Form)** в продюсерскую команду Global Generation.

**Slug**: `content-scout`
**URL**: `/s/content-scout`
**Seed**: `npx tsx prisma/seed-content-scout.ts`

---

## О вакансии

- **Позиция**: Digital Media Producer (Instagram / TikTok)
- **Возраст кандидатов**: 16–30 лет
- **Проект**: медиа-платформа вокруг личного бренда Алисы Саксон — международное образование, поступление в США, личный бренд
- **Команда**: продюсерская команда Global Generation (5 человек)
- **Контент**: Instagram Reels, TikTok, вертикальный формат

---

## Поток

1. **Hero** — анимированная welcome-страница: "Мы ищем того, кто видит тренды раньше всех"
2. **5-шаговая форма** — анкета кандидата (без NDA)
3. **Успех** — подтверждение + уведомление о связи через Telegram

## Форма (5 шагов, ~20 вопросов)

### Шаг 1: О себе
| Поле | Тип | Валидация |
|------|-----|-----------|
| Имя и фамилия | Текст | Обязательно |
| Telegram | Текст | Обязательно, авто-префикс `@`, дедупликация |
| Возраст | Число | 16–30 |
| Город | Текст | Обязательно |
| Чем занимаешься | Радио-кнопки | Школа / Вуз / Gap Year / Работа / Фриланс |

### Шаг 2: Насмотренность и аналитика
| Поле | Тип | Валидация |
|------|-----|-----------|
| 2–3 сильных аккаунта + почему | Textarea | Обязательно, 300+ символов для макс. баллов |
| Разбор вирусного ролика | Textarea | Обязательно, 250+ символов для макс. баллов |

### Шаг 3: Hard Skills
| Поле | Тип | Валидация |
|------|-----|-----------|
| Знание метрик | Чекбоксы | Retention, ER, Охваты, CTR, Среднее время, Сохранения, Репосты, Конверсия |
| Что такое хук и зачем нужен | Textarea | Обязательно, 150+ символов для макс. баллов |
| Инструменты для контента | Чекбоксы | CapCut, Canva, Figma, Adobe, Notion, Google Sheets, Другое |
| AI-инструменты | Чекбоксы | ChatGPT, Claude, Midjourney, HeyGen, Runway, Другое (необязательно) |
| Уровень английского | Радио | Начальный → Advanced+ |

### Шаг 4: Опыт и портфолио
| Поле | Тип | Валидация |
|------|-----|-----------|
| Есть ли опыт ведения аккаунтов | Да/Нет | Обязательно |
| Подробности об опыте | Textarea | Условный (если Да), 250+ символов для макс. баллов |
| Ссылка на портфолио | Текст | Необязательно |
| Идея Reels/TikTok для аккаунта про поступление | Textarea | Обязательно, 200+ символов для макс. баллов |

### Шаг 5: Мотивация и готовность
| Поле | Тип | Валидация |
|------|-----|-----------|
| Почему интересна позиция | Textarea | Обязательно, 200+ символов для макс. баллов |
| Часов в неделю | Радио | 5–10 / 10–20 / 20–30 / Full-time |
| Когда можешь начать | Радио | Сразу / Через неделю / 2 недели / Месяц |
| Откуда узнал о вакансии | Радио-кнопки | Instagram, TikTok, Telegram, Знакомые, HH, Другое (необязательно) |
| Согласие на обработку данных | Consent | Обязательно |
