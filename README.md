# GG Mentor Survey

Платформа отбора менторов по educational consulting для Global Generation.

## Стек

- **Next.js 16** (App Router) + TypeScript
- **Prisma 6** + **Neon PostgreSQL**
- **Tailwind CSS 4** + shadcn/ui
- **Docker** + AWS ECR/EC2 (деплой)

## Быстрый старт

```bash
# 1. Установить зависимости
npm install

# 2. Создать .env файл
cp .env.example .env
# Заполнить DATABASE_URL и DIRECT_URL из Neon dashboard

# 3. Применить миграции
npx prisma migrate deploy

# 4. Создать admin пользователя
npm run seed

# 5. Создать ментор-опросник в БД
npm run seed:mentor

# 6. Запустить
npm run dev
```

## Роуты

| URL | Описание |
|-----|----------|
| `/` | Главная страница |
| `/s/mentor` | Опросник для менторов |
| `/s/mentor/success` | Страница успешной отправки |
| `/admin` | Админ панель (логин/пароль) |
| `/admin/surveys/[id]/responses` | Список ответов на опросник |
| `/admin/surveys/[id]/stats` | Статистика по опроснику |

## Структура опросника

1. **О себе** — имя, Telegram, email, город, LinkedIn
2. **Профессиональный опыт** — стаж, кол-во студентов, страны, уровни образования
3. **Экспертиза** — университеты, языки, стипендии, сильные стороны
4. **Методология** — подход к студенту, работа с эссе, мотивация
5. **Тестовое задание** — кейс (стратегия поступления) + фидбек на эссе
6. **Финальный шаг** — доступность, источник, согласие
